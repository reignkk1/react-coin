## Using


- Styled-components : 컴포넌트 스타일링 , ThemeProvider 사용

- Recoil : 라이트모드 / 다크모드 구현에 사용

- React-query : API 이용해 정보 가져오는데 사용

- React-router-dom : 링크에 따른 화면전환 위해 사용

- React-apexcharts : 차트를 그리기 위해 사용
